Building Blocks of HTML

An HTML document consists of its basic building blocks, which are:

Tags

An HTML tag surrounds the content and applies meaning to it. It is written between < and > brackets.

Attributes

An attribute in HTML provides extra information about the element, and it is applied within the start tag. An HTML attribute contains two fields: name and value.

Syntax

<tag name attribute_name="attr_value"> content </tag name>

Elements

An HTML element is an individual component of an HTML file. In an HTML file, everything written within tags is termed as HTML elements.
